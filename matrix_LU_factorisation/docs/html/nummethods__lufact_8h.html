LU-Factoriser implementing Crout's method. In accordance with Crout's method, it assumes the diagonal of the upper matrix U to be 1s (ones).

Parameters
AThe source matrix, which must be square.
LThe lower triangular matrix, passed as reference to be written to.
UThe upper triangular matrix, also passed as reference. In this method, its diagonal is assumed to be 1s.
