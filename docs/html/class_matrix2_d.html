Mathod to extract a submatrix from the current matrix. The submatrix is defined by the position of its top-left element and its size (counting from that element).

pos_yx-position of the top-left element of the desired submatrix.

sub_size_yColumn count of desired submatrix.
Returns
A reference to the resulting submatrix.
Exceptions
Throwsrange_error() if the submatrix exceeds the source's range, for example when its top-left element is at the bottom-right of the source and its size is larger than 1 in any direction.
